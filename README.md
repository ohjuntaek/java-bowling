# 볼링 게임 🎳

## 요구사항

- 볼링 점수를 계산하는 프로그램 구현
- 1단계 목표 : 점수 계산을 제외한 볼링 게임 점수판을 구현
- 각 프레임?이 스트라이크면 "X"
- 스페어면 "9 | /" 과 같이 출력
- 미스면 "8|1" 과 같이 출력
  - 스트라이크 : 프레임의 첫번째 투구에서 모든 핀(10개)를 쓰러트린 상태
  - 스페어 : 프레임의 두번째 투구에서 모든 핀을 쓰러트린 상태
  - 미스 : 프레임의 두번쨰 투구에서도 모든 핀이 쓰러지지 않은 상태
  - 거터 : 핀을 하나도 쓰러트리지 못한 상태. 거터는 "-"로 표시
- 10프레임은 스트라이크거나 스페어면 한 번을 더 투구할 수 있다.

## 기능 정의

  - 스트라이크, 스페어, 미스, 거터
- 게임 진행하기
